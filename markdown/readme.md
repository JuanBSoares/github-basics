## Code

### Inline Code

You can print to the terminal using the `puts "hello world"` command.

### Multi line Code

#### Without Highlighting
```
def hello_world
    puts "hello world"
end
```

#### With Highlighting
```rb
def hello_world
    puts "hello world"
end
```
